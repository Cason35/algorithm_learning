# algorithm_learning

前端工程师算法训练营，使用 JavaScript 刷主流算法题。

## 课程周期

建议按 **35 天** 学完第一轮，每天 60-90 分钟：

- 30 天：覆盖主流算法和高频题型，可以开始稳定做简单题和部分中等题。
- 35 天：多 5 天复盘和综合训练，更适合形成题感。
- 45-60 天：如果目标是大厂面试，可以在 35 天后进入二刷和专题强化。

我的建议：先走 35 天。每一天会有：

- `class_schedule.md`：当天学习课程、原理讲解、生活例子、举一反三。
- `work.js`：当天作业，你完成后运行自测；每个作业函数会带力扣风格的 `@param` / `@return` 注释，先说明输入输出契约。
- 批改流程：你写完告诉我，我会检查代码、跑测试、指出问题，并把学习总结追加到当天的 `class_schedule.md`。

## 使用方式

运行某天作业：

```bash
npm run day1
npm run day2
npm run day3
```

或者直接：

```bash
node day1/work.js
node day2/work.js
node day3/work.js
```

第一次运行通常会失败，因为函数里留了 TODO。你完成实现后再运行，全部通过后告诉我：

```text
我写完 day1/work.js 了，帮我批改
```

我会继续作为你的算法老师检查实现质量。
