# Day 8：栈

## 今天目标

第二周从新的基础数据结构开始：栈。

栈的规则很简单：

```text
后进先出
```

也就是最后放进去的东西，最先被拿出来。JavaScript 里可以直接用数组模拟栈：

```js
const stack = [];

stack.push("A");
stack.push("B");
stack.pop(); // B
```

今天你要掌握 4 件事：

1. 栈适合处理“最近的未匹配项”。
2. 括号匹配为什么天然适合栈。
3. 如何用栈处理路径、撤销、相邻抵消。
4. 单调栈的第一层直觉。

## 1. 生活例子

想象一摞盘子。

你洗完一个盘子就放到最上面，下一次拿盘子时，也一定先拿最上面的那个。

这就是栈：

- `push`：放到最上面。
- `pop`：从最上面拿走。
- `stack[stack.length - 1]`：看最上面的元素，但不拿走。

## 2. 栈适合什么题

看到这些信号，可以想栈：

- 最近的一个还没匹配的东西。
- 成对出现，比如括号。
- 撤销操作。
- 路径回退，比如 `..`。
- 相邻元素抵消。
- 找下一个更大/更小元素。

前端里的类似场景：

- 浏览器历史记录。
- 编辑器撤销/重做。
- 解析 HTML 标签嵌套。
- 面包屑路径处理。
- AST / 表达式解析。

## 3. 括号匹配

括号匹配是栈最经典的题。

例子：

```text
({[]})
```

扫描时：

- 遇到左括号：入栈。
- 遇到右括号：看栈顶是不是对应左括号。
- 对应：弹出。
- 不对应或栈为空：非法。

扫描结束后，栈必须为空。

```js
function isValidParentheses(s) {
  const stack = [];
  const pairs = new Map([
    [")", "("],
    ["]", "["],
    ["}", "{"],
  ]);

  for (const char of s) {
    if (char === "(" || char === "[" || char === "{") {
      stack.push(char);
    } else if (pairs.has(char)) {
      if (stack.pop() !== pairs.get(char)) {
        return false;
      }
    }
  }

  return stack.length === 0;
}
```

## 4. 路径简化

Unix 路径里：

- `.` 表示当前目录。
- `..` 表示回到上一级。
- 多个 `/` 可以当成一个。

例子：

```text
/a/./b/../../c/ -> /c
```

处理方式：

1. 用 `/` 切分路径。
2. 普通目录名入栈。
3. 遇到 `..` 就弹出一个目录。
4. 遇到空字符串或 `.` 就跳过。
5. 最后用 `/` 拼回路径。

## 5. 相邻抵消

有些题里，相邻相同元素会抵消。

比如：

```text
abbaca
```

扫描：

- `a` 入栈。
- `b` 入栈。
- 下一个 `b` 和栈顶相同，弹出。
- `a` 和栈顶 `a` 相同，弹出。
- `c` 入栈。
- `a` 入栈。

结果是：

```text
ca
```

## 6. 单调栈入门

单调栈是栈的一种进阶用法：栈里元素保持某种单调性。

例题：每日温度。

给定温度数组，返回每一天之后要等几天才会有更高温度。

思路：

- 栈里存“还没找到更高温度的下标”。
- 当前温度比栈顶下标对应温度高时，说明栈顶那天找到了答案。
- 弹出栈顶，并计算天数差。

今天只做入门版，重点是理解“栈里存下标，而不是只存值”。

### 可以继续优化的点

`isValidParentheses` 里可以用一个 `Map` 存右括号到左括号的对应关系，这样不用 `includes` 和 `indexOf` 反复查数组：

```js
const pairs = new Map([
  [")", "("],
  ["]", "["],
  ["}", "{"],
]);
```

`dailyTemperatures` 里可以把栈顶下标先取出来，代码会更清楚：

```js
const prevIndex = stack.pop();
result[prevIndex] = i - prevIndex;
```

这些都是可读性优化，不影响你当前实现的正确性。

### 今天真正学到的东西

栈题要先说清楚“栈里存什么”：

- 括号题：存还没匹配的左括号。
- 路径题：存当前有效目录。
- 相邻抵消：存还没被抵消的字符。
- 单调栈：存还没找到答案的下标。

当你能说清楚栈里元素的含义，入栈和弹栈规则通常就会自然很多。
